Upload bitsyin-share-short-links folder to the `/wp-content/plugins/` directory1. Activate the plugin through the \'Plugins\' menu in WordPress1. Place `[bitsy_share]` in your templates== Frequently Asked Questions ==What sharing tools does this work with?We have tested it on the following sharing tools: Wordpress Jetpack, ShareThis, Facebook (official facebook plugin), Really simple Facebook Twitter Share.How does the plugin handle previously created posts?Posts that were created prior to installing the plugin will show up with a share and short link generator widget at the centre-right of the web page once you visit the particular page.
