With Bitsy URL shortner, you can shorten URLs to make them easier to share on social media sites.== Installation ==This section describes how to install the plugin and get it working.e.g.1. Upload bitsyin-share-short-links folder to the `/wp-content/plugins/` directory1. Activate the plugin through the \'Plugins\' menu in WordPress1. Create a page or a post and insert the shortcode [bitsy_share] into the text.1. Insert the shortcode "echo do_shortcode('[bitsy_share]');" anywhere in php code.== Screenshots ==1.
